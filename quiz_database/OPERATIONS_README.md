# Quiz Database Operations

Target port alignment:
- PostgreSQL listens on localhost:5001

Start database:
- ./startup.sh

Verify Postgres is listening:
- psql -h localhost -p 5001 -U appuser -d myapp -c "SELECT 1;"

Connection helper:
- See db_connection.txt for a ready-to-copy psql URL (generated by startup.sh)

Backups:
- ./backup_db.sh  (backs up to database_backup.sql using port 5001)

Restore:
- ./restore_db.sh (restores from database_backup.sql using port 5001)

DB Viewer (optional):
- cd db_visualizer
- source postgres.env
- node server.js           # default port 5002
- or: PORT=5050 node server.js
- Open http://localhost:5002

Notes:
- If you encounter EADDRINUSE when starting the viewer, choose a different PORT.
- All scripts assume port 5001 for Postgres.

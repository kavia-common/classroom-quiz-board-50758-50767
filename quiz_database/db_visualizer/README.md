# Simple DB Viewer

This viewer exposes lightweight endpoints to test database connectivity (PostgreSQL/MySQL/SQLite/MongoDB) and a basic UI.

Default port: 5002 (chosen to avoid conflicts with common dev ports like 3000/5000)

How to run (PostgreSQL only example):
1) Ensure the quiz database is running and listening on 5001:
   - In the quiz_database folder, run: ./startup.sh
   - Verify: psql -h localhost -p 5001 -U appuser -d myapp -c "SELECT 1;"

2) Source the PostgreSQL env for the viewer:
   - source postgres.env
   (this file is generated by startup.sh and sets POSTGRES_URL/USER/PASSWORD/DB/PORT)

3) Start the viewer on port 5002 (default) or override with PORT:
   - node server.js           # uses port 5002 by default
   - PORT=5050 node server.js # or choose another free port

Then open: http://localhost:5002

API hints:
- GET /api/databases            -> tests configured DB connections
- GET /api/postgres/tables      -> list tables (public schema)
- GET /api/postgres/tables/<t>/data?limit=50

If you see EADDRINUSE:
- Another process is already using the port. Re-run with a different port:
  PORT=5051 node server.js
